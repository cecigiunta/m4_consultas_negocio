# Consultas con Operadores de agreación COUNT, SUM, AVG, MIN, MAX

## Descripción
Este proyecto corresponde al trabajo práctico del módulo 4. 
Cuenta con 4 consultas que se resuelven utilizando operadores de agregación, CASE WHEN y subconsultas.

## Propósito del ejercicio
En esta práctica se realizan cálculos para responder a diferentes necesidades dentro de la base de datos Ventas_Tech_DB y la tabla "dbo.ventas".
Se realizan consultas con SELECT, COUNT, SUM/AVG, TOP, ORDER BY, GROUP BY, CASE WHEN y subconsultas
